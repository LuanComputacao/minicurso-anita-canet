Aqui nós aprenderemos como construir uma página utilizando:

  • HTML semântico
  • CSS estruturado (BEM)
  • JS funcional

E a linguagem definitiva de IGU para se construir sites

Esta linguagem utiliza marcadores para definir o comportamentos e conteúdos em qualquer navegador de internet

O CSS usa uma linguagem simples para definir como sua página será exibida renderizada pelo motor do Navegador de Internet

É fácil de escrever

Primeiro defina seus seletores no arquivo HTML depois sinta-se livre para definir propriedades que irão desenhar sua página

O JavaScript é a linguagem mais poderosa para definir comportamentos em sua página

Se você quer carregar mais conteúdos, o JavaScript fará

Se você quer limpar um campo de um formulário, o JavaScript irá ajudar

Se você quer mudar o estilo de um Bloco quando o usuário clicar na área dele, o JavaScript fará isso.
